# conway-go-wasm

Go + WebAssembly と Vite + React + Tailwind CSS を組み合わせて、GitHub Pages 上で動作するシンプルなライフゲームを実装します。

## 概要

このプロジェクトでは、Conway's Game of Life を題材にして、フロントエンドと WebAssembly を組み合わせた構成で実装します。

描画やUIは Vite + React + Tailwind CSS をベースに構成し、ゲームロジックは Go + WebAssembly で扱います。成果物は GitHub Pages へデプロイします。

## 技術スタック

- Go
- WebAssembly
- Vite
- React
- TypeScript
- Tailwind CSS
- GitHub Pages
